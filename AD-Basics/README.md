# TryHackMe Active Directory Basics Room
This project documents the completion of a basic Active Directory room on TryHackMe for learning and documenting purposes.
>Note: Connection made to remote machine via RDP.
## What did I learn?
- What Active Directory is
- What an Active Directory Domain is
- What components go into an Active Directory Domain
- Forests and Domain Trust
- Group Policy Objects
- Managing Users & Computers in AD
- Authentication Methods
- Delegation
- Policy Change/Creation
